## なぜ「ストラテジ系」を集中的に学ぶべきなのか

ITパスポート試験の全100問のうち、ストラテジ系は **約35問** を占めます。つまり、全体の **配点の35%** がこの分野に集中しています。

しかも、ストラテジ系は範囲が広い。企業活動の基礎から経営戦略、法務、最新のAI倫理まで、ビジネスの「常識」が問われます。

ここで重要なのは、ストラテジ系の知識は **試験のためだけのものではない** ということです。

### 実務で即戦力になる知識

- **経営戦略マネジメント** ：PPM、SWOT分析、マーケティング手法など、ビジネスの共通言語を習得できる
- **法務** ：個人情報保護法、知的財産権、労働関連法規など、社会人として知っておくべき法的知識
- **企業活動** ：財務諸表の読み方、データ分析の基礎など、どの職種でも求められるリテラシー
- **ビジネスインダストリ** ：AI、IoT、DXなど、最新テクノロジーのビジネス応用

つまり **ストラテジ系を制する者は、試験もキャリアも制する** と言っても過言ではありません。

### このドリルの特徴

このドリルには、ストラテジ系の問題を **101問** 収録しています。

| 中分類               | 問題数 |
| -------------------- | ------ |
| 企業活動             | 33問   |
| ビジネスインダストリ | 26問   |
| 経営戦略マネジメント | 20問   |
| 法務                 | 10問   |
| システム戦略         | 8問    |
| 技術戦略マネジメント | 3問    |
| システム企画         | 1問    |

### 苦手問題の優先出題アルゴリズム

普通のドリルアプリとの最大の違いは、**苦手な問題ほど出やすくなる** こと。

- 初回は **未回答の問題が優先的に** 選ばれ、まんべんなく出題される
- 間違えた問題は **次回以降のセッションで出題確率が5〜8倍** に上がる
- 正解した問題は出題確率が下がるが、ゼロにはならない（忘却防止）

つまり、繰り返すほどあなたの弱点だけを集中攻撃する **オーダーメイドのドリル** になります。

### 使い方

1. まず **30問、40問、50問** のいずれかを選んでスタート
2. 解答後すぐに正誤と **AI解説** が表示される
3. セッション終了後に **中分類ごとの正答率** を確認
4. 「もう一度挑戦」ボタンで、苦手が優先された新しいセットに挑戦
