## 専門用語の暗記を「作業」から「ハック」へ

情報処理技術者試験の合格を阻む最大の壁、それは **膨大な専門用語の暗記** です。
「問題演習は楽しいけれど、用語を覚えるのが苦痛…」そんな声に応えるために開発したのが、この **暗記ハック（Flashcards）** です。

### このアプリの特長

1.  **インプット特化のUX** : 4択問題よりも高速に、テンポよく用語をチェックできます。
2.  **最新シラバス完全準拠** : 生成AIや最新のITトレンド用語をいち早く網羅。
3.  **隙間時間の活用** : PCでもスマホでも、片手でサクサク学習が進みます。

### さあ、学習を始めましょう

<FlashcardApp client:load allData={allData} />

### 推奨される学習フロー

1.  **インプット** : このフラッシュカードで、知らない用語を「脳にインデックス」します。
2.  **仕分け** : 覚えたカードは「✓」、不安なものは「✕」で仕分け、効率的に反復します。
3.  **アウトプット** : 用語を覚えたら、[生成AI・AI倫理クイズ](/app/genai-ethics-quiz)などのドリルで実戦力を試しましょう。
