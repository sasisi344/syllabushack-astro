# Syllabus Parser

This tool helps structure the raw text from IPA Syllabus PDF into the JSON format required for `syllabusRef`.

1. Select the **Target Exam** (IP, SG, FE).
2. Select the **Middle Category** you are working on.
3. Paste the **Keywords** section from the PDF (e.g., the text column under "用語例" or "技術要素").
4. Click **Extract**.
5. Repeat for other categories.
6. Click **Copy Full JSON** to get the `syllabus-{examId}.json` content.
